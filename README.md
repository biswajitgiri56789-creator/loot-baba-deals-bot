# Loot Baba Deals Bot

This bot automatically posts **Amazon deals** to a Telegram channel every hour.

## Setup

1. Create a Telegram bot and get `BOT_TOKEN`.
2. Create a Telegram channel and get `CHANNEL_ID`.
3. Get your Amazon affiliate `AMAZON_STORE_ID`.
4. Add these as **GitHub Secrets** in your repo:
   - BOT_TOKEN
   - CHANNEL_ID
   - AMAZON_STORE_ID

## Run

The bot runs automatically via **GitHub Actions** every hour.
